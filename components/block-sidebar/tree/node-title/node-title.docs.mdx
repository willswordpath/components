import { NodeTitle } from './node-title';

A node component that is used as the title of a collapsable folder in the docs tree node.  
Besides controlling the opening of a tree node, this component is responsible for the navigation to configuration and overview sections of a docs section.

### Component usage

```js live
<NodeTitle id="My Folder" icon="dependencies-icn" configPath="config-path" open={false} setOpen={() => {}} />
```
